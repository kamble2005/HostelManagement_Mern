 # Hostel Management System

A Hostel Management System built using the MERN Stack.  
This application helps manage hostel students, rooms, and allocations efficiently.

---

## Features
- Add new students
- View student list
- Add hostel rooms
- Manage room availability
- Allocate rooms to students

---

## Tech Stack
- MongoDB (Database)
- Express.js (Backend Framework)
- React.js (Frontend)
- Node.js (Server)

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/your-username/hostel-management-system.git

2. Install backend dependencies

cd backend
npm install

3. Install frontend dependencies

cd frontend
npm install

---

##  Run Application on Portal

### Run Backend Server
cd backend
npm start

Backend will run on:
http://localhost:5000

### Run Frontend Application
cd frontend
npm start

Frontend will run on:
http://localhost:3000

Open your browser and visit the frontend portal to use the Hostel Management System.

---

## API Routes

### Students
- GET /students → Get all students
- POST /students → Add new student

### Rooms
- GET /rooms → Get all rooms
- POST /rooms → Create new room
